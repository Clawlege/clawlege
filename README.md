# 🦞 Clawlege

**Learn to build your own AI agent. Free. Open source. For everyone.**

Clawlege is an interactive learning platform that takes you from "what's an AI agent?" to "I built one and it's awesome" — no coding required for the first three modules.

Whether you're 10 or 50, a developer or someone who's never written a line of code, this course will teach you to build an AI agent that actually *does things* for you.

Built on [OpenClaw](https://openclaw.ai), the open source AI agent framework.

---

## 🚀 Getting Started

### Prerequisites
- A computer (Mac, Linux, or Windows)
- [Node.js](https://nodejs.org) (v18 or higher)
- A curious mind

### Quick Start

```bash
# 1. Clone this repo
git clone https://github.com/Clawlege/clawlege.git
cd clawlege

# 2. Install OpenClaw
npm install -g openclaw

# 3. Run the setup wizard
openclaw setup

# 4. Start Module 1
open modules/01-meet-your-agent/README.md
```

That's it. You're ready to build your first AI agent.

## 📚 The Curriculum

| Module | What You'll Build | Time | Difficulty |
|--------|-------------------|------|------------|
| [01 — Meet Your Agent](modules/01-meet-your-agent/) | Your first AI agent with a personality | ~30 min | 🟢 Beginner |
| [02 — Give It Skills](modules/02-give-it-skills/) | An agent that searches, calculates, and fetches | ~30 min | 🟢 Beginner |
| [03 — Securing Your Agent](modules/03-securing-your-agent/) | API keys, prompt injection, permissions & privacy | ~60 min | 🟢 Beginner |
| [04 — Memory & Personality](modules/04-memory-and-personality/) | An agent that remembers you | ~45 min | 🟢 Beginner |
| [05 — Connect to the World](modules/05-connect-to-the-world/) | Messages, emails, and real-time data | ~60 min | 🟡 Intermediate |
| [06 — Build Something Unique](modules/06-build-something-unique/) | Your own custom agent project | ~90 min | 🟡 Intermediate |
| [07 — Hardware Week](modules/07-hardware-week/) | Agents on phones, Raspberry Pi, and more | ~60 min | 🟡 Intermediate |
| [08 — Polish & Ship](modules/08-polish-and-ship/) | A production-ready agent you can share | ~60 min | 🔴 Advanced |
| [09 — Demo Day](modules/09-demo-day/) | Present your creation to the world | ~30 min | 🔴 Advanced |

**Modules 1-4 require zero coding.** You'll write plain English files that tell your agent who it is and how to behave. Seriously — that's all it takes.

## 🎭 Starter Templates

Don't want to start from scratch? Grab a template and customize it:

- **[Personal Assistant](templates/personal-assistant/)** — Daily briefings, reminders, calendar management
- **[Homework Helper](templates/homework-helper/)** — Research, explain, and summarize any topic
- **[Sports Tracker](templates/sports-tracker/)** — Live scores, standings, and player stats
- **[Weather Buddy](templates/weather-buddy/)** — Forecasts and outfit suggestions
- **[Recipe Finder](templates/recipe-finder/)** — Meal planning and grocery lists

Each template includes a ready-to-go `SOUL.md`, `USER.md`, and `AGENTS.md`. Copy one into your OpenClaw workspace and start chatting.

## 🤔 Why Clawlege?

Most AI tools treat you like a user. We think you should be a **builder**.

- **No gatekeeping.** Everything here is free and open source.
- **No coding required** to get started. You'll write plain English, not Python.
- **Runs locally.** Your data stays on your machine.
- **Progressive.** Start simple, go as deep as you want.
- **Fun.** Learning should feel like discovery, not homework.
