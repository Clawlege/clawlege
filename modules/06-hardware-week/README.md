# Module 6: Hardware Week

**⏱ Time: ~60 minutes | 🟡 Difficulty: Intermediate | 💻 Coding: Some**

Your agent escapes the computer.

---

## What You'll Learn

- **Agents on mobile** — Running OpenClaw as a node on your phone
- **Raspberry Pi agents** — A dedicated always-on agent for $35
- **Camera and sensors** — Agents that can see and sense the physical world
- **Location awareness** — Agents that know where you are
- **Multi-device setups** — One agent brain, multiple device endpoints

## Module Outline

### Lesson 1: The Node System
- What are OpenClaw nodes?
- Pairing your phone as a node
- Commands: camera, location, notifications
- The agent-node relationship

### Lesson 2: Phone Agent
- Setting up OpenClaw on iOS or Android
- Camera access — your agent can take photos
- Location services — context-aware responses
- Notifications — proactive alerts on your phone

### Lesson 3: Raspberry Pi Setup (Optional)
- Why a Pi is the perfect agent host ($35, always on, silent)
- Installation guide
- Running your agent 24/7
- Headless operation and remote management

### Lesson 4: Multi-Device Agent
- One brain, multiple bodies
- Routing: which device handles which task
- Practical setup: phone for camera, Pi for always-on, laptop for heavy work
- Managing nodes

## Challenge Preview
Set up your agent on at least one additional device (phone or Raspberry Pi) and demonstrate it using a hardware feature (camera, location, or sensors).

## Prerequisites
- Completed Modules 1-5
- A smartphone or Raspberry Pi (optional but recommended)
