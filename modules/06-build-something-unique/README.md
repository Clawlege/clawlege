# Module 5: Build Something Unique

**⏱ Time: ~90 minutes | 🟡 Difficulty: Intermediate | 💻 Coding: Optional**

Stop following tutorials. Start building YOUR thing.

---

## What You'll Learn

- **Project ideation** — How to come up with an agent idea that solves a real problem
- **Architecture thinking** — Planning which skills, memory, and channels your agent needs
- **Custom skills** — Writing your first skill (it's easier than you think)
- **Iteration** — The build→test→improve cycle that makes agents great
- **Portfolio piece** — By the end, you'll have something worth showing off

## Module Outline

### Lesson 1: What Should I Build?
- Start with a problem, not a technology
- The "annoyance audit" — what repetitive tasks do you hate?
- 20 project ideas across categories (school, sports, creative, family, productivity)
- Scoping: what's achievable in 90 minutes?

### Lesson 2: Plan Your Agent
- Mapping out personality, skills, and memory needs
- The Agent Design Canvas (a simple template)
- Choosing the right channels
- Writing the SOUL.md, USER.md, and AGENTS.md for your project

### Lesson 3: Custom Skills (Optional)
- What a skill looks like under the hood
- Writing a basic skill script
- Installing your custom skill
- Testing and debugging

### Lesson 4: Build, Test, Improve
- Your first prototype (it will be rough — that's fine)
- Testing with real questions
- Common pitfalls and how to fix them
- Getting feedback from a friend

## Challenge Preview
Build a custom agent from scratch that solves a real problem in your life. Document what it does, how you built it, and what you'd improve.

## Prerequisites
- Completed Modules 1-4
