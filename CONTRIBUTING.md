# Contributing to Clawlege

Thanks for wanting to make Clawlege better! Here's how to help.

## Ways to Contribute

### 🐛 Bug Reports
Found a broken link, typo, or incorrect instruction? [Open an issue](https://github.com/Clawlege/clawlege/issues) with the `bug` label.

### 📝 Content Improvements
- Fix typos or unclear explanations
- Add better examples
- Improve challenge descriptions
- Translate content to other languages

### 🎭 New Templates
Create a new starter agent template! Each template needs:
- `SOUL.md` — agent personality
- `USER.md` — user info template with placeholders
- `AGENTS.md` — operating manual

### 📚 New Modules
Modules 4-8 need full content. If you want to write one:
1. Open an issue saying which module you want to tackle
2. We'll discuss the approach
3. Submit a PR when ready

### 🎨 Landing Page
Design improvements, accessibility fixes, and mobile optimizations are always welcome.

## How to Submit

1. Fork the repository
2. Create a branch (`git checkout -b fix/typo-in-module-1`)
3. Make your changes
4. Test locally (read through the content, check links)
5. Submit a pull request

## Content Guidelines

- **Tone:** Friendly, encouraging, slightly irreverent. Like a cool older sibling.
- **Audience:** 10-year-olds to 50-year-olds. No jargon without explanation.
- **Structure:** Every lesson should have "Try it now!" moments every few paragraphs.
- **Challenges:** Should be achievable in the stated time. Include example solutions.
- **No coding** in Modules 1-3. Keep them plain English only.

## Code of Conduct

Be kind. Be helpful. Be patient. Everyone here is learning.

We don't tolerate harassment, gatekeeping, or "well actually" energy. If someone asks a "basic" question, help them — we all started somewhere.

## Questions?

[Open an issue](https://github.com/Clawlege/clawlege/issues) with the `question` label. We're happy to help.
